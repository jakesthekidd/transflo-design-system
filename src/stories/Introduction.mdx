# PrimeNG Component Library

Welcome to your design system's Storybook — built with **Angular** and **PrimeNG 21**.

This is your single source of truth for UI components. Every component shown here maps directly to what gets built in production.

---

## Component Categories

| Category | Components |
|---|---|
| **Button** | Primary, Secondary, Severity variants, Sizes, Icons, Loading |
| **Forms** | InputText, Select/Dropdown, Checkbox, RadioButton, Textarea, ToggleSwitch |
| **Data Display** | Card, Table, Badge, Tag, Avatar |
| **Overlay** | Dialog, Toast, Tooltip, Messages |
| **Navigation** | Tabs, Breadcrumb, Paginator |

---

## How to Use This Storybook

- **Browse** components in the left sidebar
- **Controls** panel (bottom) — change props live to see variants
- **Docs** tab — full API documentation per component
- **A11y** tab — accessibility audit for each component

---

## How to Add or Update Components

Tell Claude Code what you need, for example:
- *"Add a story for the PrimeNG Chip component"*
- *"Add a dark theme variant to the Button stories"*
- *"Update the Table story to show sorting"*
